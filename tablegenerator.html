This generator generates both HTML and Markdown tables. You can either choose a style from the dropdown, this will generate a class from my CSS tables, which you can find here.

Or you can press "Include Styling" that will bring up styling for your table. The colors you choose will reflect in the output when you press "Generate Table".

Also, remember to choose the number of columns and rows you want.

you are happy with your table, simply press "Copy HTML" for HTML or "Copy Markdown" for Markdown. Then paste it where you want it.

If you are going to use GoblinTheme CSS table styles, make sure to paste this link above your table:

<link rel="stylesheet" href="https://sliverspell.github.io/goblinthemes/tablestyles/tables.css">
